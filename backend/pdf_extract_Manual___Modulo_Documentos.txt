1 - INTRODUÇÃO
Este documento foi elaborado com o objetivo de orientar os usuários do E-Palmas
quanto à operacionalização do Módulo de Documentos. Este manual apresenta os procedimentos
operacionais que permitirão a utilização plena deste recurso na Instituição.
Para facilitar a interação sistema-usuário, este manual demonstra a composição das
funcionalidades e ações do Módulo de Documentos e a facilidade da navegabilidade e usabilidade
por meio de descrições sucintas dos procedimentos e dos resultados que serão alcançados.
O manual do Módulo de Documentos descreve as funcionalidades disponibilizadas no
sistema para a gestão dos documentos em situação corrente. Ações como cadastro, inclusão de
arquivos digitais, exclusão, tramitação e recebimento de documentos são contempladas neste
módulo.
O acesso às funcionalidades deste módulo é definido pelos perfis atribuídos aos
usuários. Algumas ações são permitidas apenas aos administradores/gestores do sistema.

2 - MÓDULO DOCUMENTOS
O MÓDULO DOCUMENTOS é destinado às funcionalidades de gerenciamento de
documentos (protocolos) pelo sistema. Os documentos no E-Palmas são como pastas virtuais
onde estarão inseridos os arquivos digitais. São identificados no NUP (Número Único de
Protocolo) pelo número .9. localizado logo antes do número sequencial final, o que o difere do
documento que apresenta .0. logo antes do número sequencial final.
Para navegar no Módulo Documentos o usuário deve posicionar o cursor do mouse sobre
o respectivo menu do módulo. O sistema habilitará a relação de funcionalidades disponíveis; o
usuário deverá clicar sobre a funcionalidade que deseja operar e o sistema o direcionará para lá.

2.6 - Cancelar Tramitação
A funcionalidade Cancelar Tramitação é destinada ao desfazimento da última
tramitação sofrida por um documento. Uma tramitação só pode ser cancelada pelo
departamento que a realizou.
Procedimentos:
▪ O sistema listará todos os documentos tramitados pelo departamento do usuário e
ainda não recebidos pelo departamento de destino;
▪ Selecionar, entre os documentos listados, aqueles cuja(s) tramitação(ões) deseja
cancelar;
▪ Acionar o botão ‘CANCELAR’;
▪ O sistema emitirá uma mensagem como a apresentada a seguir em
confirmação ao sucesso da ação.
A tramitação de um documento só pode ser cancelada antes do efetivo recebimento pelo
departamento de destino.

2.10 – Emprestar
A funcionalidade Emprestar é destinada ao empréstimo de documentos em
situação de ‘ARQUIVADO’ ou ‘SOBRESTADO. Por esta funcionalidade também são
registradas as devoluções dos documentos emprestados.
O empréstimo registra que o documento foi retirado do arquivo do departamento
para consulta por um usuário de outro departamento. O empréstimo é diferente da
tramitação, ele não altera o departamento de carga do documento, apenas sua situação.
Procedimentos:
Emprestar documento:
▪ Informar o número do documento que deseja emprestar;
▪ Acionar o botão ‘PESQUISAR’;
▪ Acionar o botão +, na opção ‘NÚMEROS ADICIONAIS’, para emprestar mais de um
documento ao mesmo tempo;
▪ Selecionar o departamento do usuário para o qual o documento será emprestado;
▪ Selecionar o usuário para o qual o(s) documento(s) será(ão) emprestado(s);
▪ Indicar a data limite para o retorno do documento, se necessário;

atendimento emitidos pelo departamento;

2.28 - Tramitar
A funcionalidade Tramitar é destinada à tramitação (movimentação) de
documentos de um departamento para outro no sistema.
Procedimentos:
▪ Informar o número do documento que deseja alterar;
▪ Acionar o botão ‘PESQUISAR’;
▪ Acionar o botão +, na opção ‘NÚMEROS ADICIONAIS’, para adicionar mais
documentos à tramitação;
▪ Informar se o(s) documento(s) tramitado(s) possui prioridade de tratamento no
departamento de destino;
▪ Informar o departamento de destino do(s) documento(s);
▪ Informar o motivo pelo qual;
▪ Informar um despacho acerca da tramitação do(s) documento(s);
